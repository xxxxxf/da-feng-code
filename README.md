# Hbuilder X工具构建 基于Color-ui

#### 本地运行教程 微信小程序端
1. 下载 HBuilder X (App 开发版)、微信小程序开发工具
2. HBuilder X 中安装 scss 插件 工具 -> 插件安装 -> scss
3. 微信小程序开发工具中开启服务端口  设置 -> 安全设置 -> 服务端口
4. 配置服务端域名及验证参数
5. 运行 -> 运行到小程序模拟器
6. 如果没有开发者权限，需要管理员添加

#### 测试服运行教程
1. 前三步同本地环境
2. 配置服务端域名及验证参数 
3. 发行 -> 自定义发行 -> 测试环境小程序
4. 在微信开发者工具中点击上传至微信服务器
5. 在小程序后台选择需要的版本为体验版

#### 正式服运行教程
1. 前三步同本地环境
2. 配置服务端域名及验证参数
3. 发行 -> 小程序-微信
4. 在微信开发者工具中点击上传至微信服务器
5. 在小程序后台选择需要的版本提交审核
6. 审核通过之后便可发布
